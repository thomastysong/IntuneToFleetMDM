# Security policy

## Reporting a vulnerability

Please do **not** open a public GitHub issue for security reports.

Instead, contact the maintainer privately (GitHub profile email/contact), and include:

- What you found
- Repro steps
- Impact assessment

## Sensitive data

This project deals with Windows MDM enrollment and may run on managed endpoints.

Do not share or commit:

- NuGet API keys
- Fleet API tokens
- Orbit node keys
- Device identifiers that are sensitive in your environment
